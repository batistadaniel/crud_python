# 🚀 CRUD Python + MySQL

## 📦 Instalação

Instale a dependência necessária:

pip install mysql-connector-python

---

## 🗄️ Configuração do MySQL (Workbench)

1. Abra o MySQL Workbench  
2. Em MySQL Connections, clique no ícone de +  
3. Preencha os dados:

- Connection Name: (nome de sua preferência)  

    ### Matenha os dados abaixo:
- Connection Method: Standard (TCP/IP)  
- Hostname: 127.0.0.1  
- Port: 3306  
- Username: root  

4. Clique em Test Connection  
5. Informe a senha: senac  

---

## 📋 Executar script SQL

1. Copie o conteúdo do arquivo database.sql  
2. Cole no editor do MySQL Workbench  
3. Execute todos os comandos com:

Ctrl + Shift + Enter  (Para rodar tudo de uma só vez) [Icone de raio a esquerda] 

Ou

Ctrl + Enter (Para rodar cada trecho de SQL separadamente) [Icone de raio a direita] 

---

## ▶️ Executar o projeto

1. Abra o projeto no VSCode  
2. Abra o terminal  
3. Execute o comando:

python main.py
